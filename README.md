# shoutbot

Easy, lightweight HTTP to desktop notification service for macOS (and possibly others)

## Usage

```
POST /notify

{
    "title": "Notification title",
    "message": "Notification message",
    "sound": true
    "icon": "myicon.png"
}
```

See config-example.js for configuration.
